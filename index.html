Some mods are only allowed in set seed runs and will cause your run to be rejected if used in other categories. Currently, these mods are Set Spawn and ChunkCacher.

Mods that have legality warnings within them have the force of law and must be followed. Currently, this includes Seedqueue's Show Debug Menu option and Set Spawn's message in chat if incorrectly configured.

If you choose to use Extra Options or Dynamic FOV off you are required to follow A.8.12 of the full rules.

OptiFine is legal for 1.14.4 and below under very specific conditions. You must read A.8.1 of the full rules before using it as the default settings are not legal (specifically Smooth World and Lazy Chunk Loading, in versions where those options are present), and many other of its other settings are banned. OptiFine is not allowed for glitched or oneshot runs.

All other mods not listed as allowed are banned under the strictest terms, resulting in immediate run rejection. This includes Fabric API, a library mod often used outside of MCSR. None of the allowed mods depend on Fabric API. This also includes [MCSR Ranked], which is for playing head-to-head matches, not for leaderboard speedruns.

All mods use the [Fabric Loader]. This can be set up legally in multiple ways, see [this guide]. Only a handful of launchers verified to not significantly modify the game are currently allowed, these are:

  • [Vanilla launcher]
  • [MCSR Launcher] (recommended)
  • [Prism Launcher]
  • [MultiMC]
  • [Modrinth App]
  • [ATLauncher]
Versions 1.3 to 1.13.2 use Legacy Fabric mappings. Currently the only legal way to install these is with MCSR Launcher or the MCSR-specific [MultiMC / Prism Launcher instances]. DO NOT USE the install methods from the Legacy Fabric project themselves as they modify game libaries. Versions 1.0.0 to 1.2.5 use OrnitheMC mappings, with the same install options as Legacy Fabric. Unfortunately, Modrinth modpacks currently do not support either Legacy Fabric or OrnitheMC.

Please note that previously legal builds of mods are often made illegal upon updates for closer vanilla parity or bug fixes, DO NOT assume that just because the mods you have installed were legal at the time they are still legal now. A grace period of a month to update mods is typically given for parity updates, this is often shorter for bug fixes. When using mods you have a responsibility to ensure they are updated. Important changes should be posted in the [speedrun.com/mc news tab] but we encourage you to join the [MCSR Discord server aka. Javacord] as that is the most consistent way to get annoucements.
